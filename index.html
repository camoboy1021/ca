Tools for FiveM and Unturned

FiveM Tools

Resource Creator
Generate template resources for your FiveM servers fast and easily.

This tool lets you create __resource.lua and proper folder structures for your scripts.
Visit: FiveM Docs

Vehicle Spawn Code Generator
Easily generate vehicle spawn codes for custom cars on your FiveM server.

Input car models and receive spawn codes, which you can use directly in your scripts or server console.

Unturned Tools

Asset ID Finder
Look up item, vehicle, and weapon IDs for Unturned quickly.

Allows you to search the full Unturned item database and copy IDs
Reference: Unturned Item Database

Permission Command Generator
Generate permissions commands for server admins and plugins easily.

Choose a role, permissions, and auto-generate commands to paste into your commands.dat or via in-game chat.
